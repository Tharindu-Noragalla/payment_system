❓ Quick Help — v2
Phase 1 — Full frontend UI with live template editor and 5 themes.
🎨 Print Template Editor:
Go to Print Templates. Select a template, use − + buttons to nudge any field position. Canvas updates live. Save when done.
🏦 Bank Account → Template:
In Bank Accounts → Edit, assign a print template to each bank account. The correct template loads automatically when printing.
🎨 Themes:
Use the 5 colour dots in the top bar, or go to Settings → Appearance. Preference is saved in browser.
🔄 Workflow rule:
Always TICK checkboxes before Transfer / Accept / Issue. Highlighted row alone does NOT process.
